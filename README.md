# Bài 11: Máy tính
Viết chương trình máy tính đơn giản:
Đầu tiên, xuất hiện Menu: 1. Để sử dụng máy tính; 2. Thoát
Nếu chọn 1: chương trình xử lý như sau
- "Nhập số thứ nhất": 3
- "Nhập số thứ hai": 7
Mời bạn chọn phép tính: "(+, -, *, :)": +
Kết quả là:
3.0 + 7.0 = 10.0
"Mời bạn chọn 1 để tiếp tục và chọn 2 để thoát"
